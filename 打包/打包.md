 # MeterSphere 项目打包文档

## 环境要求

- JDK 17+
- Maven 3.6+（推荐优先使用仓库自带 `./mvnw`）
- Node.js v20.8.1 与 npm（前端打包）
- Docker（如需构建镜像）

## 打包步骤

### 1. 初始化依赖（首次必做）

```bash
cd /Users/edy/ideaProjects/metersphere

# 仅安装父 POM，不递归子模块
./mvnw install -N

# 构建共享 SDK 与基础依赖
./mvnw clean install -pl framework,framework/sdk-parent,framework/sdk-parent/domain,framework/sdk-parent/sdk,framework/sdk-parent/xpack-interface,framework/sdk-parent/jmeter
```

### 2. 后端打包（全部模块）

```bash
./mvnw clean package -DskipTests -pl framework/eureka,framework/gateway,system-setting/backend,project-management/backend,test-track/backend,api-test/backend,performance-test/backend,report-stat/backend,workstation/backend,workflow-service/backend,analytics-stat/backend -am
```

## 本地启动（全部后端服务）

说明：每个服务建议单独开一个终端窗口。

```bash
# 1) 基础服务
./mvnw spring-boot:run -f framework/eureka/pom.xml
./mvnw spring-boot:run -f framework/gateway/pom.xml

# 2) 业务服务
./mvnw spring-boot:run -f system-setting/backend/pom.xml
./mvnw spring-boot:run -f project-management/backend/pom.xml
./mvnw spring-boot:run -f test-track/backend/pom.xml
./mvnw spring-boot:run -f api-test/backend/pom.xml
./mvnw spring-boot:run -f performance-test/backend/pom.xml
./mvnw spring-boot:run -f report-stat/backend/pom.xml
./mvnw spring-boot:run -f workstation/backend/pom.xml
./mvnw spring-boot:run -f workflow-service/backend/pom.xml
./mvnw spring-boot:run -f analytics-stat/backend/pom.xml
```

## 常见问题

- 若你只使用系统 Maven，请把 `./mvnw` 替换为 `mvn`。
- 启动前请先确认 MySQL、Redis 等依赖服务可用。
- 某模块启动失败时，优先单独执行该模块 `clean package` 再启动。
