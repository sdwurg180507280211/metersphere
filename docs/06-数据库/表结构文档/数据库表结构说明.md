# MeterSphere 数据库表结构说明

## 一、系统基础表

### 用户和组织管理
- **user** - 用户信息表，存储系统用户的基本信息
- **user_role** - 用户角色关联表，记录用户在不同范围内的角色
- **role** - 角色表，定义系统、组织、工作空间等不同层级的角色
- **organization** - 组织表，存储组织信息
- **workspace** - 工作空间表，存储工作空间信息
- **project** - 项目表，存储项目基本信息
- **user_group** - 用户组表，用于用户分组管理
- **user_group_permission** - 用户组权限表，记录用户组的权限配置

### 文件管理
- **file_metadata** - 文件元数据表，存储文件的名称、类型、大小等元信息
- **file_content** - 文件内容表，存储文件的二进制内容

### 系统配置
- **system_parameter** - 系统参数表，存储系统配置参数
- **test_resource_pool** - 测试资源池表，管理测试执行资源池
- **test_resource** - 测试资源表，存储具体的测试资源配置

## 二、测试跟踪（Test Track）模块

### 测试用例相关
- **test_case** - 测试用例主表，存储测试用例的完整信息（名称、类型、优先级、步骤等）
- **test_case_node** - 测试用例节点表，用于组织测试用例的树形结构
- **test_case_template** - 测试用例模板表，存储测试用例模板配置
- **custom_field_test_case** - 测试用例自定义字段值表，存储测试用例的自定义字段值
- **test_case_follow** - 测试用例关注表，记录用户关注的测试用例

### 测试计划相关
- **test_plan** - 测试计划表，存储测试计划的基本信息
- **test_plan_test_case** - 测试计划用例关联表，关联测试计划和测试用例
- **test_plan_node** - 测试计划节点表，组织测试计划的树形结构
- **test_plan_report** - 测试计划报告表，存储测试计划执行报告
- **test_plan_report_content** - 测试计划报告内容表，存储报告的详细内容
- **test_plan_report_data** - 测试计划报告数据表，存储报告统计数据
- **test_plan_follow** - 测试计划关注表，记录用户关注的测试计划
- **test_plan_principal** - 测试计划负责人表，记录测试计划的负责人

### 测试评审相关
- **test_case_review** - 测试用例评审表，存储评审任务信息
- **test_case_review_test_case** - 评审用例关联表，关联评审和测试用例
- **test_case_review_test_case_users** - 评审用例用户关联表，记录评审用例的评审人
- **test_case_review_node** - 测试用例评审节点表，组织评审的树形结构
- **test_case_review_follow** - 测试用例评审关注表，记录用户关注的评审

### 缺陷管理
- **issues** - 缺陷表，存储缺陷的基本信息
- **test_case_issues** - 测试用例缺陷关联表，关联测试用例和缺陷
- **custom_field_issues** - 缺陷自定义字段值表，存储缺陷的自定义字段值
- **issue_follow** - 缺陷关注表，记录用户关注的缺陷

### 报告相关
- **test_case_report** - 测试用例报告表，存储测试用例执行报告
- **test_case_report_template** - 测试用例报告模板表，存储报告模板配置

## 三、API 测试模块

### API 定义
- **api_definition** - API 定义表，存储 API 接口的定义信息
- **api_module** - API 模块表，组织 API 的模块结构
- **api_test_case** - API 测试用例表，存储 API 测试用例
- **api_definition_exec_result** - API 定义执行结果表，存储 API 执行结果
- **api_definition_env** - API 定义环境表，存储 API 的环境配置
- **api_definition_follow** - API 定义关注表，记录用户关注的 API
- **api_test_case_follow** - API 测试用例关注表，记录用户关注的 API 测试用例

### API 场景
- **api_scenario** - API 场景表，存储 API 测试场景
- **api_scenario_reference_id** - API 场景引用ID表，记录场景间的引用关系
- **api_scenario_follow** - API 场景关注表，记录用户关注的场景

### API 测试
- **api_test** - API 测试表，存储 API 测试任务
- **api_test_file** - API 测试文件关联表，关联测试和文件
- **api_test_report** - API 测试报告表，存储 API 测试报告
- **api_test_report_detail** - API 测试报告详情表，存储报告的详细内容

### API 文档
- **api_document_share** - API 文档分享表，存储 API 文档分享信息

## 四、性能测试模块

### 性能测试
- **load_test** - 性能测试表，存储性能测试任务配置
- **load_test_file** - 性能测试文件关联表，关联测试和文件
- **load_test_follow** - 性能测试关注表，记录用户关注的性能测试

### 性能测试报告
- **load_test_report** - 性能测试报告表，存储性能测试报告基本信息
- **load_test_report_detail** - 性能测试报告详情表，存储报告的详细数据
- **load_test_report_log** - 性能测试报告日志表，存储测试执行日志
- **load_test_report_result** - 性能测试报告结果表，存储测试结果数据
- **load_test_report_result_part** - 性能测试报告结果分片表，分片存储大量结果数据
- **load_test_report_result_realtime** - 性能测试报告实时结果表，存储实时测试数据

## 五、UI 测试模块

- **ui_scenario** - UI 场景表，存储 UI 自动化测试场景
- **ui_scenario_module** - UI 场景模块表，组织 UI 场景的模块结构
- **ui_scenario_report** - UI 场景报告表，存储 UI 测试报告
- **ui_scenario_report_detail** - UI 场景报告详情表，存储报告的详细内容
- **ui_scenario_report_result** - UI 场景报告结果表，存储测试结果
- **ui_scenario_report_structure** - UI 场景报告结构表，存储报告的结构信息
- **ui_element** - UI 元素表，存储 UI 页面元素信息
- **ui_element_reference** - UI 元素引用表，记录元素的引用关系
- **ui_scenario_reference** - UI 场景引用表，记录场景间的引用关系
- **ui_task_refresh** - UI 任务刷新表，记录 UI 任务的刷新状态

## 六、自定义字段

- **custom_field** - 自定义字段表，定义各种场景下的自定义字段
- **custom_field_template** - 自定义字段模板表，存储字段模板配置
- **custom_field_test_case** - 测试用例自定义字段值表，存储测试用例的自定义字段值
- **custom_field_issues** - 缺陷自定义字段值表，存储缺陷的自定义字段值

## 七、其他功能表

### 集成相关
- **service_integration** - 服务集成表，存储第三方服务集成配置

### 通知相关
- **notification** - 通知表，存储系统通知信息

### 关系图
- **relationship_edge** - 关系边表，存储各种实体间的关系图数据

### 插件相关
- **plugin** - 插件表，存储插件配置信息

### 环境相关
- **api_test_environment** - API 测试环境表，存储 API 测试环境配置
- **environment_group** - 环境组表，组织环境的分组
- **environment_group_project** - 环境组项目关联表，关联环境组和项目

### 其他
- **test_case_test** - 测试用例测试关联表，关联测试用例和其他测试
- **attachment** - 附件表，存储附件信息
- **attachment_module_relation** - 附件模块关联表，关联附件和各个模块
- **minder_extra_node** - 脑图额外节点表，存储脑图的额外节点数据
- **custom_function** - 自定义函数表，存储自定义函数定义
- **report_statistics** - 报告统计表，存储报告统计数据

---

**说明：**
- 所有表使用 utf8mb4 字符集
- 主键 ID 通常为 varchar(50)
- 时间字段使用 bigint(13) 存储时间戳（毫秒）
- 关注表（follow）用于记录用户对各类资源的关注关系
- 自定义字段值表采用 key-value 方式存储，支持不同数据类型的字段值
