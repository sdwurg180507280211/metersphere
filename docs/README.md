## 📝 文档规范

### 命名规范
- 文件名使用中文，语义清晰
- 技术术语保持英文（如 Maven、Docker、JVM）
- 使用连字符分隔多个概念

### 文档结构
每个文档应包含：
1. 标题和简介
2. 背景/问题描述
3. 解决方案/实现方式
4. 示例代码（如适用）
5. 注意事项
6. 参考资料

### 代码示例
- 添加详细中文注释
- 说明关键逻辑和设计思路
- 标注版本兼容性
