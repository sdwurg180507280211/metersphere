# 龙族幻想故事

## 第1章：归来

路明非睁开眼睛的时候，首先看到的是一片白色。

不是那种纯净的雪白，而是医院特有的、带着消毒水味道的苍白。天花板上的日光灯管发出嗡嗡的声响，像是有只困兽在金属牢笼里挣扎。他动了动手指，感觉到某种冰凉的液体正顺着血管流进身体。

"你醒了。"

声音从右边传来。路明非缓慢地转过头，看到诺诺坐在床边的椅子上。她看起来糟透了——红色的长发乱糟糟地披散着，眼睛下面有浓重的黑眼圈，嘴唇干裂得像是久旱的土地。她手里握着一杯咖啡，但咖啡早就凉透了，表面结了一层薄薄的膜。

"我……"路明非想要说话，但喉咙干涩得像是有砂纸在摩擦。他咳嗽了几声，诺诺立刻放下杯子，从床头柜上拿起一瓶水，拧开盖子递到他嘴边。

"慢点喝。"

水很凉，带着一丝甜味。路明非贪婪地吞咽着，感觉那股凉意一直流到胃里，让他混沌的意识稍微清醒了一些。

"我们在哪？"他问。

"卡塞尔学院的医疗部。"诺诺说，"你已经昏迷了三天。"

三天。路明非在心里默念这个数字。三天前发生了什么？他的记忆像是一卷被水浸泡过的胶片，模糊不清。他记得西伯利亚的暴风雪，记得末日派的地下基地，记得那个巨大的尼伯龙根……

然后他想起来了。

路鸣泽。

那个总是穿着黑色小西装、叫他"哥哥"的男孩。那个在梦境中陪伴他无数次的存在。那个……最后选择了牺牲自己的……

"路鸣泽呢？"路明非猛地坐起来，动作太猛让他一阵眩晕。他抓住诺诺的手腕，"他在哪？"

诺诺没有回答。她的眼神闪烁了一下，然后移开了视线。

"诺诺，告诉我，他在哪？"

"明非……"诺诺的声音很轻，像是在怕惊扰什么，"他……不在了。"

"什么叫不在了？"

"他消失了。"诺诺说，"在你昏迷之后，我们找到了你。你一个人躺在雪地里，身边……什么都没有。"

路明非松开她的手。他低头看着自己的双手——苍白、瘦弱、带着输液针头的痕迹。这是他的手，路明非的手，那个废柴的手。

但他能感觉到有什么不一样了。

在他的身体深处，有什么东西正在流动。不是血液，不是某种可以名状的物质，而是一种……存在感。像是有人在他的灵魂里点燃了一盏灯，微弱但持久。

"他给了我什么。"路明非喃喃道。

"什么？"

"路鸣泽。"路明非抬起头，看着诺诺的眼睛，"他在消失之前，给了我什么东西。我能感觉到。"

诺诺愣住了。她仔细地端详着路明非的脸，然后她的表情变了——从困惑变成惊讶，从惊讶变成某种难以言喻的复杂情绪。

"你的眼睛……"她说。

"我的眼睛怎么了？"

诺诺没有回答。她站起身，从床头柜的抽屉里拿出一面小镜子，递给他。

路明非接过镜子，看向里面的自己。

还是那个熟悉的脸——平凡得不能再平凡的五官，乱糟糟的黑色短发，因为长期缺乏睡眠而显得有些憔悴。但他的眼睛……

他的眼睛是金色的。

不是龙类那种暴虐的金色，也不是路鸣泽那种带着戏谑的金色。这是一种更温暖的颜色，像是黄昏时分的阳光，又像是秋天落叶的颜色。

"这是什么……"路明非伸手触碰自己的眼睛，但镜子里的倒影也跟着做同样的动作。

"从你醒来开始就是这样。"诺诺说，"医生检查过，说你的眼睛没有任何异常。但……"

她停顿了一下。

"但什么？"

"但昂热校长说，这是'世界树的痕迹'。"

世界树。路明非在心里默念这个词。他想起了那个梦境——那个巨大的、贯穿天地的巨树，黑王尼德霍格正在啃食它的根部。他想起了路鸣泽最后说的话：

"我们是世界树的两面，哥哥。生命与死亡，创造与毁灭。"

"校长在哪？"路明非问。

"他在英灵殿。"诺诺说，"学院……出了很多事。"

路明非注意到她说"英灵殿"而不是"校长室"。他想起昏迷前看到的景象——卡塞尔学院在战火中燃烧，英灵殿的穹顶坍塌，图书馆的书架倒塌成一片废墟。

"学院怎么了？"

诺诺沉默了很久。当她再次开口时，声音里带着一丝颤抖：

"末日派发动了袭击。在你进入尼伯龙根之后，他们派出了龙化死侍军团。我们……损失了很多。"

"很多人死了？"

"是的。"

"谁？"

诺诺没有回答。她只是静静地看着他，眼神里有悲伤，有怜悯，还有一种路明非读不懂的东西。

"告诉我，诺诺。"路明非说，"我有权利知道。"

"芬格尔……"诺诺说，"他在掩护学生撤离的时候受了重伤，现在还在重症监护室。"

路明非感到心脏被什么东西揪紧了。芬格尔，那个总是吊儿郎当、自称"废柴"的师兄。那个在关键时刻却从不退缩的人。

"还有呢？"

"凯撒……"诺诺的声音更低了，"他为了救我，被龙息正面击中。医生说……他可能再也醒不过来了。"

路明非闭上眼睛。

凯撒·加图索，那个金发的高大男孩，总是带着贵族式的傲慢和骄傲。他是诺诺的未婚夫，是路明非的情敌，也是……他的朋友。

"还有吗？"

"零失踪了。"诺诺说，"战斗结束后，没有人找到她。有人说看到她走进了火场，但搜救队什么都没有发现。"

零。那个神秘的俄罗斯女孩，总是面无表情，总是跟在路鸣泽身边。她和路鸣泽之间有什么联系？路明非从未问过，但现在他想知道。

"楚子航呢？"他问。

诺诺的表情变得更加复杂。

"他……"她说，"他醒过来了。"

"那不是很好吗？"

"但他的记忆……"诺诺停顿了一下，"他的记忆停留在了十五岁。他不记得之后发生的任何事，包括……"

"包括什么？"

"包括夏弥。"

路明非沉默了。

夏弥。大地与山之王耶梦加得，也是楚子航爱过的女孩。她在北京地铁站的尼伯龙根里死在了楚子航的刀下，而楚子航一直背负着这份痛苦。

现在他忘记了。这不是解脱，这是另一种惩罚。

"我要见他。"路明非说。

"现在不行。"诺诺摇头，"校长下令，在你完全恢复之前，不能见任何人。"

"为什么？"

"因为……"诺诺咬了咬嘴唇，"因为你现在的状态很不稳定。你的龙血活性超过了正常值的三倍，但你的意识完全清醒。这在混血种历史上从未有过。校长担心……"

"担心什么？"

"担心你已经成为某种……超越混血种的存在。"

路明非笑了。那笑容里有苦涩，有自嘲，还有一种说不出的疲惫。

"所以他们把我当成实验品了？"

"不。"诺诺说，"他们把你当成希望。"

她站起身，走到窗边。窗外是卡塞尔学院的庭院，但现在那里只剩下断壁残垣。樱花树被连根拔起，喷泉池里积满了黑色的灰烬。

"末日派不是唯一的威胁。"诺诺说，"在战斗结束后，我们收到了来自世界各地的情报。龙类正在大规模苏醒，比任何人预料的都要快。"

"新的龙王？"

"不仅仅是龙王。"诺诺转过身，金色的阳光从窗外照进来，在她的红发上跳跃，"是某种更古老的东西。校长说，诸神黄昏的预言正在应验。"

诸神黄昏。路明非想起那个梦境，想起黑王啃食世界树根部时发出的满足的叹息。

"路鸣泽……"他喃喃道，"他消失之前，说过一些话。"

"什么话？"

"他说……我们还会再见面的。在某个地方，某个时间。也许是以另一种形式。"

诺诺沉默了。她走回床边，重新坐下，握住了路明非的手。

"明非。"她说，"有件事我必须告诉你。"

"什么？"

"在你昏迷的时候……"诺诺深吸一口气，"有人看到你身边出现了一些奇怪的现象。"

"什么现象？"

"金色的光。"诺诺说，"从你身体里散发出来的。那些光在空中凝聚，形成了……某种形状。"

"什么形状？"

诺诺看着他，眼神里有恐惧，但更多的是敬畏。

"一棵树。"她说，"一棵金色的、发光的树。它的枝叶覆盖了整个房间，然后……"

"然后什么？"

"然后它消失了。"诺诺说，"但在消失之前，有人听到了声音。"

"什么声音？"

诺诺没有回答。她只是静静地看着他，然后轻声说：

"有人在叫你的名字。哥哥。"

路明非感到心脏停止了跳动。

路鸣泽。那是路鸣泽的声音。即使在消失之后，即使在世界之外的某个地方，他仍然在叫他。

"他还活着。"路明非说，声音沙哑但坚定，"我知道他还活着。"

"明非……"

"我要找到他。"路明非说，"不管他在哪里，不管他变成了什么。我要找到他。"

他掀开被子，想要下床，但腿一软差点摔倒。诺诺赶紧扶住他。

"你现在不能乱动！"她说，"你的身体还需要恢复！"

"我等不了了。"路明非说，"每过一分钟，他就可能离我更远。"

"但你连他在哪里都不知道！"

"我知道。"路明非说。他抬起手，放在自己的胸口。在那里，他能感觉到某种微弱的脉动——不是心跳，而是某种更深层的、与这个世界相连的节奏。

"他在我心里。"路明非说，"或者说，我在他心里。我们是相连的，诺诺。从出生的那一刻起，我们就从未真正分离过。"

诺诺看着他，眼神复杂。

"你变了，明非。"她说。

"什么？"

"以前的你……"诺诺停顿了一下，"以前的你不会说这种话。以前的你会先吐槽，会想要逃避，会说'这种事怎么可能轮到我'。"

路明非愣了一下，然后笑了。

"你说得对。"他说，"我确实变了。"

他走到窗边，看着外面的废墟。夕阳正在落下，把天空染成金红色。那种颜色让他想起了路鸣泽的眼睛。

"但我不得不变。"他说，"因为有人为我付出了代价。如果我还像以前那样废柴，那他的牺牲就毫无意义了。"

诺诺走到他身边，并肩站着。

"我会帮你的。"她说。

"什么？"

"我说，我会帮你的。"诺诺转过头，看着他，"不管你要做什么，不管你要去哪里。我会陪着你。"

路明非看着她。那双深红色的眼睛里有一种他从未见过的坚定。

"为什么？"他问。

"因为你救过我。"诺诺说，"在东京，在高天原，在无数个地方。现在轮到我了。"

路明非想要说什么，但门突然被推开了。

一个穿着白大褂的医生走进来，看到路明非站在窗边，愣了一下。

"路明非先生，您应该躺在床上休息……"

"我没事了。"路明非说，"我要见校长。"

"但是……"

"告诉他。"路明非说，金色的眼睛在夕阳中闪闪发亮，"告诉他，路明非回来了。真正的路明非。"

医生被他眼中的光芒震慑住了，愣了几秒，然后点点头，转身离开了。

房间里重新安静下来。路明非和诺诺站在窗边，看着夕阳一点点沉下去。

"明非。"诺诺突然说。

"嗯？"

"你的眼睛……"她伸出手，轻轻触碰他的眼角，"在发光。"

路明非看向窗户的倒影。在玻璃的反光中，他看到自己的眼睛——那双金色的瞳孔正在发出微弱的光芒，像是两颗小小的太阳。

而在那光芒的深处，他似乎看到了什么。

一个身影。一个小小的、穿着黑色西装的身影。他站在一片金色的光芒中，背对着路明非，正在走向某个遥远的地方。

"路鸣泽……"路明非喃喃道。

那个身影似乎听到了他的声音，停下脚步，转过身来。

他看不清那个身影的脸，但他能感觉到——那个身影在微笑。

"等着我。"路明非轻声说，"我一定会找到你。"

光芒消失了。路明非的眼睛恢复了正常的金色，不再发光。

但他知道，那不是幻觉。

路鸣泽还在某个地方等着他。而他，路明非，会不惜一切代价找到他。

因为那是他的弟弟。

永远都是。

---

窗外，最后一缕阳光消失了。夜幕降临，星星开始在天空中闪烁。

而在某个遥远的地方，某个不属于这个世界的维度，一棵金色的巨树正在缓缓生长。它的根系延伸到无数个世界，它的枝叶覆盖了整个苍穹。

在树的根部，一个穿着黑色西装的男孩正坐在那里，晃荡着双腿。

"快点来吧，哥哥。"他轻声说，"我等你很久了。"

他抬起头，看向无尽的虚空。在那里，有一颗星星正在闪烁——那是路明非所在的世界。

"但这一次，"路鸣泽笑了，"换我来找你了。"
