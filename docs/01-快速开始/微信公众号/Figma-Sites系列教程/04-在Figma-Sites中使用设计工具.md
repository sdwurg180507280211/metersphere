# 🔥 Figma Sites 设计工具详解：打造专业网站！

Figma Sites 虽然是无代码工具，但设计功能一点都不差！这篇文章带你玩转 Figma Sites 里的设计工具，让你的网站看起来更专业！

---

## 一、先搞懂：Figma Sites 的设计哲学

### 简单理解

**Figma Sites = 设计师熟悉的 Figma 体验 + 网站发布能力**

如果你用过 Figma Design，那么 Figma Sites 的界面会让你感觉很亲切！

### Figma Sites 的设计原则

1. **简单优先**
   - 不需要记住几百个快捷键
   - 常用功能一目了然

2. **网站友好**
   - 所有设计都是为了"变成网站"
   - 自动考虑响应式、可访问性

3. **灵活但不混乱**
   - 有足够的设计自由度
   - 但不会让你做出"无法实现"的设计

## 二、核心设计工具详解

### 1. 选择工具（Select）

#### 用途
- 选中元素
- 移动元素
- 调整大小

#### 快捷键
- `V`（和 Figma Design 一样！）

#### 技巧
- 点击元素选中
- 按住 `Shift` 多选
- 双击进入组/Section 编辑
- 按 `Cmd/Ctrl + A` 全选

### 2. 文本工具（Text）

#### 用途
- 添加标题
- 添加段落
- 添加列表

#### 文本设置

| 设置项 | 说明 | 选项 |
|-------|------|------|
| **字体** | 文字样式 | 多种系统字体 |
| **字号** | 文字大小 | 12px - 96px |
| **字重** | 粗细 | Light、Regular、Medium、Bold |
| **对齐** | 左右中对齐 | 左、中、右 |
| **颜色** | 文字颜色 | 色板或自定义 |
| **行高** | 行间距 | 1.0 - 2.0 |
| **字间距** | 字间距 | -5% - 20% |

#### 文本类型建议

| 用途 | 字号 | 字重 |
|-----|------|------|
| 主标题 | 48-64px | Bold |
| 副标题 | 24-32px | Medium |
| 正文 | 16-18px | Regular |
| 小字 | 12-14px | Regular |

### 3. 形状工具（Shape）

#### 可用形状

| 形状 | 用途 |
|-----|------|
| **矩形** | 按钮、卡片背景、分隔线 |
| **圆形** | 头像、装饰元素、图标背景 |
| **线** | 分隔线、装饰 |

#### 形状设置

| 设置项 | 说明 |
|-------|------|
| **填充** | 内部颜色（纯色、渐变） |
| **描边** | 边框颜色和粗细 |
| **圆角** | 圆角半径（0-100px） |
| **效果** | 阴影、模糊 |

#### 圆角参考值

| 效果 | 圆角值 |
|-----|-------|
| 锐利 | 0-4px |
| 现代 | 8-12px |
| 柔和 | 16-24px |
| 圆形 | 50%（宽高相等时）|

### 4. 图片工具（Image）

#### 支持的格式
- JPG/JPEG
- PNG（支持透明）
- WebP（推荐，文件小）
- SVG（矢量图）

#### 图片设置

| 设置项 | 说明 |
|-------|------|
| **填充方式** | Fill、Cover、Contain |
| **圆角** | 圆角半径 |
| **阴影** | 投影效果 |
| **透明度** | 0-100% |

#### 图片填充方式

| 方式 | 说明 | 适用场景 |
|-----|------|---------|
| **Fill** | 填满容器，可能裁剪 | 背景图、不想留空白 |
| **Cover** | 保持比例，完全覆盖 | 产品图、头像 |
| **Contain** | 保持比例，完整显示 | Logo、图标 |

#### 图片最佳实践

1. **尺寸合适**
   - 不要上传 10MB 的大图
   - 建议最大宽度 1920px
   - 用 WebP 格式减小文件体积

2. **Retina 支持**
   - 图片尺寸是显示尺寸的 2 倍
   - 例如：显示 400x400，图片 800x800

3. **有描述文字**
   - 给图片加 Alt Text
   - 帮助视障用户理解
   - 对 SEO 也有好处

### 5. 颜色工具

#### Figma Sites 的颜色系统

| 颜色类型 | 说明 |
|---------|------|
| **主色（Primary）** | 品牌色，用于按钮、链接 |
| **次色（Secondary）** | 辅助色，用于强调 |
| **中性色（Neutral）** | 文字、背景、边框 |
| **功能色（Status）** | 成功、警告、错误 |

#### 推荐配色方案

**方案 1：简约商务（适合官网）**
```
主色：#1890FF（蓝色）
次色：#52C41A（绿色）
文字：#262626（深灰）
背景：#FFFFFF（白色）
边框：#D9D9D9（浅灰）
```

**方案 2：现代活泼（适合产品）**
```
主色：#722ED1（紫色）
次色：#F5222D（红色）
文字：#1A1A1A（深灰）
背景：#FAFAFA（浅灰）
边框：#E8E8E8（浅灰）
```

**方案 3：极简设计（适合作品集）**
```
主色：#000000（黑色）
次色：#666666（深灰）
文字：#111111（黑色）
背景：#FFFFFF（白色）
边框：#EEEEEE（浅灰）
```

#### 颜色工具技巧

1. **用拾色器**
   - 点击颜色面板的吸管
   - 可以拾取屏幕上任意颜色

2. **保存常用颜色**
   - 点击颜色面板的 "+"
   - 把常用颜色存成色板

3. **用渐变**
   - 选择渐变选项
   - 可以做线性渐变或径向渐变
   - 适合做背景或按钮

### 6. 布局工具

#### 间距（Spacing）

Figma Sites 建议使用 8px 栅格系统：

| 间距名称 | 数值 | 用途 |
|---------|------|------|
| XS | 4px | 紧密元素 |
| S | 8px | 小间距 |
| M | 16px | 默认间距 |
| L | 24px | 大间距 |
| XL | 32px | 超大间距 |
| 2XL | 48px | 区块间距 |
| 3XL | 64px | 大区块间距 |

#### 对齐（Alignment）

| 对齐方式 | 图标 | 用途 |
|---------|------|------|
| 左对齐 | ← | 正文、列表 |
| 居中对齐 | ⬌ | 标题、英雄区域 |
| 右对齐 | → | 价格、日期 |
| 顶部对齐 | ↑ | 卡片内容 |
| 垂直居中 | ↕ | 图标+文字 |
| 底部对齐 | ↓ | 卡片底部 |

#### 分布（Distribution）

当选中多个元素时，可以用分布工具：
- 水平均匀分布
- 垂直均匀分布
- 按间距分布

### 7. 效果工具

#### 阴影（Shadow）

| 阴影类型 | 效果 | 参数示例 |
|---------|------|---------|
| **小阴影** | 轻微悬浮 | X:0, Y:2, Blur:4, Color: rgba(0,0,0,0.1) |
| **中阴影** | 明显悬浮 | X:0, Y:4, Blur:12, Color: rgba(0,0,0,0.15) |
| **大阴影** | 强烈悬浮 | X:0, Y:8, Blur:24, Color: rgba(0,0,0,0.12) |

#### 模糊（Blur）

| 模糊类型 | 效果 | 数值 |
|---------|------|------|
| **轻微模糊** | 柔和 | 4px |
| **中等模糊** | 背景虚化 | 12px |
| **强烈模糊** | 毛玻璃效果 | 24px |

#### 透明度（Opacity）

| 透明度 | 效果 | 用途 |
|-------|------|------|
| 100% | 完全不透明 | 主要内容 |
| 70-80% | 轻微透明 | 次要文字 |
| 40-50% | 半透明 | 装饰元素 |
| 20-30% | 很淡 | 背景装饰 |

## 三、实战：设计一个现代卡片

让我们用上面的工具，设计一个漂亮的产品卡片！

### 目标

```
┌─────────────────────┐
│                     │
│     [产品图片]      │
│                     │
│  产品名称           │
│  产品描述           │
│                     │
│  ¥99  [立即购买]    │
└─────────────────────┘
```

### 步骤

#### 步骤 1：创建卡片容器

1. 添加 Section Block
2. 设置宽度：320px
3. 设置背景：白色
4. 设置圆角：12px
5. 添加阴影：X:0, Y:4, Blur:12, rgba(0,0,0,0.1)

#### 步骤 2：添加产品图片

1. 添加 Image Block
2. 上传产品图片
3. 设置填充方式：Cover
4. 设置圆角：8px（顶部）
5. 设置高度：200px

#### 步骤 3：添加产品名称

1. 添加 Text Block
2. 输入产品名称
3. 设置字号：20px
4. 设置字重：Bold
5. 设置颜色：#1A1A1A
6. 设置上边距：16px
7. 设置左右边距：16px

#### 步骤 4：添加产品描述

1. 添加 Text Block
2. 输入产品描述
3. 设置字号：14px
4. 设置字重：Regular
5. 设置颜色：#666666
6. 设置行高：1.5
7. 设置上边距：8px
8. 设置左右边距：16px

#### 步骤 5：添加价格和按钮

1. 添加一个水平容器
2. 左边加价格文字：
   - 字号：24px
   - 字重：Bold
   - 颜色：#1890FF
3. 右边加按钮：
   - 文字："立即购买"
   - 背景：#1890FF
   - 文字颜色：白色
   - 圆角：8px
   - 内边距：8px 16px
4. 设置这个区域的上边距：16px
5. 设置左右边距：16px
6. 设置下边距：16px

#### 步骤 6：优化响应式

1. 切换到 Tablet 视图
   - 检查布局是否正常
2. 切换到 Mobile 视图
   - 把卡片改成全宽（100%）
   - 调整间距

#### 步骤 7：预览

点击 Preview，看看效果！

## 五、常见问题

### Q: 可以用自定义字体吗？

A: 付费版支持自定义 Web Fonts，免费版使用系统字体。系统字体已经足够漂亮了！

### Q: 可以导入 Figma Design 的组件吗？

A: 可以通过 Library 方式共享组件，详见上一篇文章。

### Q: 怎么让设计在所有设备上都好看？

A: 记住三个原则：
1. 用相对单位（%、em）代替固定像素
2. 在三个断点都检查一下
3. 保持间距一致（8px 栅格）

### Q: 可以用 CSS 吗？

A: 可以！用 Code Layer 添加自定义 CSS。

## 六、最佳实践

### ✅ 应该做的

1. **保持简单**
   - 颜色不超过 3-4 种
   - 字体不超过 2 种
   - 间距保持一致

2. **用留白**
   - 不要把元素挤得太满
   - 留白让内容更易读
   - 留白让网站更高级

3. **建立对比**
   - 标题大，正文小
   - 主要内容醒目，次要内容淡化
   - 按钮颜色突出

4. **保持一致**
   - 同样的按钮用同样的样式
   - 同样的间距重复使用
   - 同样的颜色反复出现

### ❌ 不应该做的

1. **不要加太多效果**
   - 阴影不要太重
   - 动画不要太夸张
   - 渐变不要太花哨

2. **不要用太多种颜色**
   - 彩虹色看起来不专业
   - 保持配色简单统一

3. **不要忽略响应式**
   - 不要只在 Desktop 设计
   - 记得检查手机端

4. **不要像素级纠结**
   - Figma Sites 是流式布局
   - 视觉上舒服就行
