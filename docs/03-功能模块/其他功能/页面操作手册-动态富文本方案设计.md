# 页面操作手册 - 动态富文本方案设计

## 需求分析

### 核心需求

1. **支持富文本内容**：可以显示图片、格式化文本、链接等
2. **动态配置**：从系统参数表获取内容，无需重新发版
3. **页面独立**：每个页面显示各自的帮助内容

### 技术要点

- 富文本渲染（HTML + 图片）
- 后端接口获取配置
- 前端组件封装
- 缓存优化

```java
package io.metersphere.controller;

import io.metersphere.dto.PageHelpDTO;
import io.metersphere.service.PageHelpService;
import jakarta.annotation.Resource;
import org.springframework.web.bind.annotation.*;

/**
 * 页面帮助接口
 */
@RestController
@RequestMapping("/page-help")
public class PageHelpController {
    
    @Resource
    private PageHelpService pageHelpService;
    
    /**
     * 获取指定页面的帮助内容
     * @param pageName 页面路由名称，如：test-case-list
     * @return 帮助内容（HTML格式）
     */
    @GetMapping("/{pageName}")
    public PageHelpDTO getPageHelp(@PathVariable String pageName) {
        return pageHelpService.getPageHelp(pageName);
    }
    
    /**
     * 获取所有页面帮助列表（用于管理界面）
     */
    @GetMapping("/list")
    public List<PageHelpDTO> listPageHelp() {
        return pageHelpService.listPageHelp();
    }
    
    /**
     * 保存或更新页面帮助内容（管理员功能）
     */
    @PostMapping("/save")
    @RequiresPermissions(PermissionConstants.SYSTEM_SETTING_READ_EDIT)
    public void savePageHelp(@RequestBody PageHelpDTO pageHelp) {
        pageHelpService.savePageHelp(pageHelp);
    }
    
    /**
     * 删除页面帮助内容
     */
    @DeleteMapping("/{pageName}")
    @RequiresPermissions(PermissionConstants.SYSTEM_SETTING_READ_DELETE)
    public void deletePageHelp(@PathVariable String pageName) {
        pageHelpService.deletePageHelp(pageName);
    }
}
```

### 3. Service - 业务逻辑

创建 `framework/sdk-parent/sdk/src/main/java/io/metersphere/service/PageHelpService.java`：

```java
package io.metersphere.service;

import io.metersphere.base.domain.SystemParameter;
import io.metersphere.base.domain.SystemParameterExample;
import io.metersphere.base.mapper.SystemParameterMapper;
import io.metersphere.dto.PageHelpDTO;
import jakarta.annotation.Resource;
import org.apache.commons.lang3.StringUtils;
import org.springframework.cache.annotation.CacheEvict;
import org.springframework.cache.annotation.Cacheable;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.List;
import java.util.stream.Collectors;

/**
 * 页面帮助服务
 */
@Service
@Transactional(rollbackFor = Exception.class)
public class PageHelpService {
    
    @Resource
    private SystemParameterMapper systemParameterMapper;
    
    private static final String PAGE_HELP_PREFIX = "page.help.";
    private static final String PAGE_HELP_TYPE = "page_help";
    
    /**
     * 获取页面帮助内容（带缓存）
     */
    @Cacheable(value = "pageHelp", key = "#pageName")
    public PageHelpDTO getPageHelp(String pageName) {
        String paramKey = PAGE_HELP_PREFIX + pageName;
        SystemParameter param = systemParameterMapper.selectByPrimaryKey(paramKey);
        
        if (param == null) {
            // 返回默认提示
            return PageHelpDTO.builder()
                .pageName(pageName)
                .content("<p>暂无帮助内容</p>")
                .build();
        }
        
        return PageHelpDTO.builder()
            .pageName(pageName)
            .content(param.getParamValue())
            .build();
    }
    
    /**
     * 获取所有页面帮助列表
     */
    public List<PageHelpDTO> listPageHelp() {
        SystemParameterExample example = new SystemParameterExample();
        example.createCriteria()
            .andParamKeyLike(PAGE_HELP_PREFIX + "%")
            .andTypeEqualTo(PAGE_HELP_TYPE);
        example.setOrderByClause("sort ASC");
        
        List<SystemParameter> params = systemParameterMapper.selectByExample(example);
        
        return params.stream()
            .map(param -> PageHelpDTO.builder()
                .pageName(param.getParamKey().replace(PAGE_HELP_PREFIX, ""))
                .content(param.getParamValue())
                .sort(param.getSort())
                .build())
            .collect(Collectors.toList());
    }
    
    /**
     * 保存或更新页面帮助内容
     */
    @CacheEvict(value = "pageHelp", key = "#pageHelp.pageName")
    public void savePageHelp(PageHelpDTO pageHelp) {
        String paramKey = PAGE_HELP_PREFIX + pageHelp.getPageName();
        
        SystemParameter param = systemParameterMapper.selectByPrimaryKey(paramKey);
        
        if (param == null) {
            // 新增
            param = new SystemParameter();
            param.setParamKey(paramKey);
            param.setParamValue(pageHelp.getContent());
            param.setType(PAGE_HELP_TYPE);
            param.setSort(pageHelp.getSort() != null ? pageHelp.getSort() : 0);
            systemParameterMapper.insert(param);
        } else {
            // 更新
            param.setParamValue(pageHelp.getContent());
            if (pageHelp.getSort() != null) {
                param.setSort(pageHelp.getSort());
            }
            systemParameterMapper.updateByPrimaryKey(param);
        }
    }
    
    /**
     * 删除页面帮助内容
     */
    @CacheEvict(value = "pageHelp", key = "#pageName")
    public void deletePageHelp(String pageName) {
        String paramKey = PAGE_HELP_PREFIX + pageName;
        systemParameterMapper.deleteByPrimaryKey(paramKey);
    }
}
```

## 图片处理方案

### 方案A：Base64 内嵌（推荐）

**优点**：
- 简单，无需额外存储
- 内容完整，不依赖外部资源

**缺点**：
- 数据库字段较大
- 不适合大图片

**实现**：

```html
<!-- 直接在富文本中使用Base64 -->
<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..." />
```

### 方案B：文件服务器存储

**优点**：
- 数据库字段小
- 支持大图片
- 可以复用图片

**缺点**：
- 需要文件服务器
- 依赖外部资源

**实现**：

1. 上传图片到MinIO或本地文件系统
2. 返回图片URL
3. 在富文本中引用URL

```html
<img src="/api/file/page-help/image/test-case-demo.png" />
```
