### 2. 新增标签字段的列表（添加字段+排序）

#### 2.1 缺陷管理--关联测试用例列表

**文件：** `test-track/frontend/src/business/issue/TestCaseIssueList.vue`

**修改内容：**

1. **导入依赖**（第90-91行）：
```javascript
import MsSingleTag from "metersphere-frontend/src/components/new-ui/MsSingleTag";
import { getTagToolTips, parseColumnTag } from "@/business/case/test-case";
```

2. **注册组件**（第100行）：
```javascript
components: {
  // ...
  MsSingleTag,
},
```

3. **添加标签列**（第39-66行）：
```vue
<ms-table-column
  :label="$t('commons.tag')"
  prop="tags"
  sortable="custom"
  min-width="180"
  :show-overflow-tooltip="false"
>
  <template v-slot:default="scope">
    <el-tooltip class="item" effect="dark" placement="top" v-if="scope.row.tags && scope.row.tags.length > 0">
      <div v-html="getTagToolTips(scope.row.tags)" slot="content"></div>
      <div class="oneLine">
        <ms-single-tag
          v-for="(itemName, index) in parseColumnTag(scope.row.tags)"
          :key="index"
          type="success"
          effect="plain"
          :show-tooltip="
            scope.row.tags.length === 1 && itemName.length * 12 <= 100
          "
          :content="itemName"
          style="margin-left: 0px; margin-right: 2px"
        />
      </div>
    </el-tooltip>
    <span v-else>-</span>
  </template>
</ms-table-column>
```

**关键修复点：**
- 添加了 `v-if="scope.row.tags && scope.row.tags.length > 0"` 条件判断，避免空标签导致的显示异常
- 当标签为空时显示 `-`
- 添加了 `.oneLine` CSS 样式，确保标签在一行内显示

4. **添加工具方法**（第256-262行）：
```javascript
getTagToolTips(tags) {
  return getTagToolTips(tags);
},
parseColumnTag(tags) {
  return parseColumnTag(tags);
},
```

5. **添加 CSS 样式**：
```css
.oneLine {
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
}
```

## 技术实现细节

### 标签显示逻辑

- **多标签显示**：当标签数量 > 1 时，显示第一个标签 + "+N" 的形式
- **单标签显示**：直接显示完整标签内容
- **Tooltip提示**：鼠标悬停时显示所有标签的完整内容

### 工具函数说明

**来源：** `test-track/frontend/src/business/case/test-case.js`

1. **getTagToolTips(tags)**
   - 功能：将标签数组转换为逗号分隔的字符串，用于 tooltip 显示
   - 返回：`"标签1,标签2,标签3"`

2. **parseColumnTag(tags)**
   - 功能：解析标签数组，用于列表显示
   - 返回：
     - 多标签：`["标签1", "+2"]`
     - 单标签：`["标签1"]`
