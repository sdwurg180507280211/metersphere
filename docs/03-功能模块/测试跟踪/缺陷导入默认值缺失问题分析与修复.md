# 缺陷导入默认值缺失问题分析与修复

## 问题描述

**缺陷导入时，对于模版中配置了默认值且必填的字段，如果Excel中未填写该字段，系统会将其值设为null，而不是使用模版配置的默认值。**

### 问题场景

1. 在缺陷模版中配置了一个自定义字段
2. 该字段设置为**必填**
3. 该字段配置了**默认值**（如：优先级=P2）
4. 用户导入Excel时，未填写该字段
5. **预期**：使用模版默认值P2
6. **实际**：字段值为null，违反必填约束

### 影响范围

- 所有通过Excel导入的缺陷
- 所有配置了默认值且必填的自定义字段
- 导致数据不完整，可能影响后续统计和查询

## 问题根源分析

### 代码位置

`test-track/backend/src/main/java/io/metersphere/excel/listener/IssueExcelListener.java`

### 问题代码

```java
/**
 * 构建自定义字段
 * 第565行开始
 */
private void buildFields(IssueExcelData issueExcelData, IssuesUpdateRequest issuesUpdateRequest) {
    if (MapUtils.isEmpty(issueExcelData.getCustomData())) {
        return; // ❌ 问题1：如果Excel没有任何自定义字段数据，直接返回
    }
    
    Boolean addFlag = issueExcelData.getAddFlag();
    List<CustomFieldResourceDTO> addFields = new ArrayList<>();
    List<CustomFieldResourceDTO> editFields = new ArrayList<>();
    List<CustomFieldItemDTO> requestFields = new ArrayList<>();
    
    // 将customFields转为Map，key=字段名称
    Map<String, List<CustomFieldDao>> customFieldMap = customFields.stream()
        .collect(Collectors.groupingBy(CustomFieldDao::getName));
    
    // ❌ 问题2：只处理Excel中存在的字段
    issueExcelData.getCustomData().forEach((k, v) -> {

            if (CollectionUtils.isNotEmpty(customFieldDaoList) && !customFieldDaoList.isEmpty()) {
                CustomFieldDao customFieldDao = customFieldDaoList.get(0);
                String type = customFieldDao.getType();

                            CustomFieldType.MULTIPLE_SELECT.getValue(), CustomFieldType.CHECKBOX.getValue())) {
                        customFieldItemDTO.setValue("[]");
                        customFieldResourceDTO.setValue("[]");
                    } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_INPUT.getValue())) {
                        customFieldItemDTO.setValue(Collections.emptyList());
                        customFieldResourceDTO.setValue("[]");
                    } else {
                        customFieldItemDTO.setValue(StringUtils.EMPTY);
                        customFieldResourceDTO.setValue(StringUtils.EMPTY);
                    }
                }
                // ... 其他处理逻辑
            }
        } catch (Exception e) {
            MSException.throwException(e.getMessage());
        }
    });
    
    // ❌ 问题4：没有补充Excel中缺失但模版中配置了默认值的字段

```

### 核心问题

1. **只处理Excel中存在的字段**：`issueExcelData.getCustomData()` 只包含Excel中有值的列
2. **未从模版获取默认值**：没有遍历模版的所有字段，检查哪些字段有默认值
3. **空值处理不当**：Excel中为空时，直接设为空字符串，而不是使用默认值
4. **缺少默认值补充逻辑**：没有为Excel中缺失的必填字段补充默认值

### 对比正常新增缺陷

正常通过UI新增缺陷时，前端会：

1. 加载模版配置
2. 对于有默认值的字段，自动填充默认值
3. 用户可以修改或保持默认值
4. 提交时包含所有必填字段

但导入时，**Excel中不存在的字段不会被处理**，导致默认值丢失。

## 修复方案

### 方案设计

在 `buildFields()` 方法中补充逻辑：

1. **遍历模版的所有字段**（而不是只遍历Excel中的字段）
2. **检查每个字段是否在Excel中存在**
3. **如果不存在且有默认值**：
   - 使用模版配置的默认值
   - 添加到 `addFields` 或 `editFields` 中
4. **如果Excel中存在但为空且有默认值**：
   - 使用模版配置的默认值（而不是空字符串）

### 修复代码

```java
/**
 * 构建自定义字段（修复版）
 * 
 * 修复内容：
 * 1. 补充Excel中缺失但模版中配置了默认值的字段
 * 2. Excel中为空时，优先使用模版默认值
 * 3. 确保必填字段不会为null
 */
private void buildFields(IssueExcelData issueExcelData, IssuesUpdateRequest issuesUpdateRequest) {
    Boolean addFlag = issueExcelData.getAddFlag();
    List<CustomFieldResourceDTO> addFields = new ArrayList<>();
    List<CustomFieldResourceDTO> editFields = new ArrayList<>();
    List<CustomFieldItemDTO> requestFields = new ArrayList<>();
    
    // 将customFields转为Map，key=字段名称
    Map<String, List<CustomFieldDao>> customFieldMap = customFields.stream()
        .collect(Collectors.groupingBy(CustomFieldDao::getName));
    
    // ✅ 修复1：遍历模版的所有字段（而不是只遍历Excel中的字段）
    for (CustomFieldDao customFieldDao : customFields) {
        try {
            String fieldName = customFieldDao.getName();
            String type = customFieldDao.getType();
            String defaultValue = customFieldDao.getDefaultValue();
            Boolean required = customFieldDao.getRequired();
            
            // 获取Excel中该字段的值
            Object excelValue = null;
            if (MapUtils.isNotEmpty(issueExcelData.getCustomData())) {
                excelValue = issueExcelData.getCustomData().get(fieldName);
            }
            
            // ✅ 修复2：判断是否需要使用默认值
            boolean useDefaultValue = false;
            if (excelValue == null || StringUtils.isEmpty(excelValue.toString())) {
                // Excel中未填写或为空
                if (StringUtils.isNotEmpty(defaultValue)) {
                    // 模版配置了默认值，使用默认值
                    useDefaultValue = true;
                    excelValue = defaultValue;
                } else if (BooleanUtils.isTrue(required)) {
                    // 必填但没有默认值，设置为空（后续校验会报错）
                    excelValue = StringUtils.EMPTY;
                }
            }
            
            // 如果Excel中没有该字段且没有默认值，跳过
            if (excelValue == null) {
                continue;
            }
            
            // 构建字段DTO
            CustomFieldResourceDTO customFieldResourceDTO = new CustomFieldResourceDTO();
            customFieldResourceDTO.setFieldId(customFieldDao.getId());
            
            CustomFieldItemDTO customFieldItemDTO = new CustomFieldItemDTO();
            BeanUtils.copyBean(customFieldItemDTO, customFieldDao);
            
            // ✅ 修复3：根据字段类型和值，正确设置字段值
            String finalValue = excelValue.toString();
            if (StringUtils.isEmpty(finalValue)) {
                // 空值处理
                if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_MEMBER.getValue(),
                        CustomFieldType.MULTIPLE_SELECT.getValue(), CustomFieldType.CHECKBOX.getValue())) {
                    customFieldItemDTO.setValue("[]");
                    customFieldResourceDTO.setValue("[]");
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_INPUT.getValue())) {
                    customFieldItemDTO.setValue(Collections.emptyList());
                    customFieldResourceDTO.setValue("[]");
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.RADIO.getValue(),
                        CustomFieldType.RICH_TEXT.getValue(), CustomFieldType.SELECT.getValue(),
                        CustomFieldType.FLOAT.getValue(), CustomFieldType.DATE.getValue(),
                        CustomFieldType.DATETIME.getValue(), CustomFieldType.INPUT.getValue())) {
                    customFieldItemDTO.setValue(StringUtils.EMPTY);
                    customFieldResourceDTO.setValue(StringUtils.EMPTY);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.TEXTAREA.getValue())) {
                    customFieldItemDTO.setValue(StringUtils.EMPTY);
                }
            } else {
                // 有值处理（包括默认值）
                if (StringUtils.equalsAnyIgnoreCase(type,
                        CustomFieldType.RICH_TEXT.getValue(), CustomFieldType.TEXTAREA.getValue())) {
                    customFieldItemDTO.setValue(finalValue);
                    customFieldResourceDTO.setTextValue(finalValue);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.FLOAT.getValue())) {
                    customFieldItemDTO.setValue(Float.parseFloat(finalValue));
                    customFieldResourceDTO.setValue(finalValue);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_SELECT.getValue(),
                        CustomFieldType.CHECKBOX.getValue())) {
                    if (!finalValue.contains("[")) {
                        finalValue = "[\"" + finalValue + "\"]";
                    }
                    String parseStr = parseOptionText(customFieldDao.getOptions(), finalValue);
                    customFieldItemDTO.setValue(parseStr);
                    customFieldResourceDTO.setValue(parseStr);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.SELECT.getValue(),
                        CustomFieldType.RADIO.getValue())) {
                    String parseStr = parseOptionText(customFieldDao.getOptions(), finalValue);
                    customFieldItemDTO.setValue(parseStr);
                    customFieldResourceDTO.setValue("\"" + parseStr + "\"");
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_INPUT.getValue())) {
                    if (!finalValue.contains("[")) {
                        finalValue = "[\"" + finalValue + "\"]";
                    }
                    customFieldItemDTO.setValue(finalValue);
                    customFieldResourceDTO.setValue(finalValue);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_MEMBER.getValue())) {
                    if (!finalValue.contains("[")) {
                        finalValue = "[\"" + finalValue + "\"]";
                    }
                    customFieldItemDTO.setValue(finalValue);
                    customFieldResourceDTO.setValue(finalValue);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.CASCADING_SELECT.getValue())) {
                    if (!finalValue.contains("[")) {
                        finalValue = "[\"" + finalValue + "\"]";
                    }
                    String parseStr = parseCascadingOptionText(customFieldDao.getOptions(), finalValue);
                    customFieldItemDTO.setValue(parseStr);
                    customFieldResourceDTO.setValue(parseStr);
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.DATETIME.getValue())) {
                    Date vDate = DateUtils.parseDate(finalValue, "yyyy-MM-dd HH:mm:ss");
                    finalValue = DateUtils.format(vDate, "yyyy-MM-dd'T'HH:mm");
                    customFieldItemDTO.setValue(finalValue);
                    customFieldResourceDTO.setValue("\"" + finalValue + "\"");
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.ASSOCIATED_SYSTEM.getValue())) {
                    String systemId = parseAssociatedSystemValue(finalValue);
                    customFieldItemDTO.setValue(systemId);
                    customFieldResourceDTO.setValue("\"" + systemId + "\"");
                } else if (StringUtils.equalsAnyIgnoreCase(type, CustomFieldType.MULTIPLE_ASSOCIATED_SYSTEM.getValue())) {
                    if (!finalValue.contains("[")) {
                        finalValue = "[\"" + finalValue + "\"]";
                    }
                    String parseStr = parseMultipleAssociatedSystemValue(finalValue);
                    customFieldItemDTO.setValue(parseStr);
                    customFieldResourceDTO.setValue(parseStr);
                } else {
                    customFieldItemDTO.setValue(finalValue);
                    customFieldResourceDTO.setValue("\"" + finalValue + "\"");
                }
            }
            
            // 添加到对应列表
            if (addFlag) {
                addFields.add(customFieldResourceDTO);
            } else {
                editFields.add(customFieldResourceDTO);
            }
            requestFields.add(customFieldItemDTO);
            
        } catch (Exception e) {
            LogUtil.error("处理字段失败: " + customFieldDao.getName(), e);
            MSException.throwException(e.getMessage());
        }
    }
    
    // 设置到请求对象
    if (addFlag) {
        issuesUpdateRequest.setAddFields(addFields);
    } else {
        issuesUpdateRequest.setEditFields(editFields);
    }
    issuesUpdateRequest.setRequestFields(requestFields);
}
```

### 关键修复点

1. **遍历模版所有字段**：`for (CustomFieldDao customFieldDao : customFields)`
2. **检查Excel中是否存在**：`issueExcelData.getCustomData().get(fieldName)`
3. **使用默认值**：`if (StringUtils.isNotEmpty(defaultValue)) { useDefaultValue = true; }`
4. **确保必填字段有值**：即使没有默认值，也设置为空字符串（后续校验会报错）
