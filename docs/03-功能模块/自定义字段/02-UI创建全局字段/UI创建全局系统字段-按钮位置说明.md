## 📍 详细说明

### 1. 创建字段（原有按钮）

**位置**：右上角，搜索框右侧  
**类型**：主按钮（蓝色）  
**图标**：无  
**文本**：`创建字段` / `Create Field`  
**权限**：所有有权限的用户都能看到  
**功能**：创建项目级自定义字段

```javascript
样式：
<el-button type="primary" size="small">创建字段</el-button>
```

---

### 2. 创建系统字段（新增按钮）⭐

**位置**：右上角，"创建字段"按钮右侧（相距10px）  
**类型**：警告按钮（橙色）  
**图标**：`el-icon-s-flag`（旗帜图标）  
**文本**：`创建系统字段` / `Create System Field`  
**权限**：**仅系统管理员可见**  
**功能**：创建全局系统字段

```javascript
样式：
<el-button 
  v-if="isSystemAdmin"
  type="warning" 
  icon="el-icon-s-flag" 
  size="small"
  style="margin-left: 10px;">
  创建系统字段
</el-button>
```

## 🎬 操作流程

### 方式1：点击"创建系统字段"按钮（新增）⭐

```
1. 以admin账号登录
   ↓
2. 项目设置 → 模版管理 → 自定义字段
   ↓
3. 看到右上角橙色按钮"创建系统字段"
   ↓
4. 点击按钮
   ↓
5. 弹出创建对话框（默认已勾选"全局系统字段"）
   ↓
6. 填写字段信息
   - 字段名称：手机
   - 使用场景：缺陷（默认）
   - 字段类型：文本框
   - 字段范围：✅ 全局系统字段（已默认勾选）
   ↓
7. 点击"确定"
   ↓
8. 创建成功！所有项目立即可见
```

### 方式2：点击"创建字段"按钮（原有，保留）

```
1. 以任意有权限的账号登录
   ↓
2. 项目设置 → 模版管理 → 自定义字段
   ↓
3. 点击"创建字段"按钮
   ↓
4. 弹出创建对话框
   ↓
5. 如果是admin：
   - 看到"字段范围"选项
   - 可以选择"项目字段"或"全局系统字段"
   
   如果是普通用户：
   - 看不到"字段范围"选项
   - 只能创建项目字段
   ↓
6. 填写信息，点击"确定"
```

## 💡 设计理念

### 为什么有两个按钮？

1. **创建字段**（蓝色）- 常规操作
   - 大多数用户的日常操作
   - 创建项目级字段
   - 保持原有使用习惯

2. **创建系统字段**（橙色+图标）- 特殊操作
   - 管理员专属功能
   - 更醒目的视觉提示
   - 一键创建全局字段（无需切换选项）

### 设计优势

✅ **职责分离**：常规功能 vs 管理功能  
✅ **视觉区分**：蓝色（常规） vs 橙色（警告/重要）  
✅ **权限清晰**：按钮级别的权限控制  
✅ **操作便捷**：一键直达，减少步骤  
✅ **向下兼容**：保留原有"创建字段"按钮和流程

## 🔧 技术实现

### 按钮代码

```vue
<template v-slot:header>
  <ms-table-header 
    :condition.sync="condition" 
    @search="getCustomFields" 
    @create="handleCreate"
    :create-tip="$t('custom_field.create')">
    
    <!-- ⭐ 新增：创建系统字段按钮 -->
    <template v-slot:button>
      <el-button 
        v-if="isSystemAdmin"
        type="warning" 
        icon="el-icon-s-flag" 
        size="small" 
        @click="handleCreateGlobalField"
        style="margin-left: 10px;">
        {{ $t('custom_field.create_global_system_field') }}
      </el-button>
    </template>
  </ms-table-header>
</template>

<script>
export default {
  data() {
    return {
      isSystemAdmin: false,
    }
  },
  created() {
    this.checkSystemAdmin();
  },
  methods: {
    checkSystemAdmin() {
      let user = getCurrentUser();
      if (user && user.roles) {
        this.isSystemAdmin = user.roles.some(role => 
          role.id === 'admin' || role.name === 'admin'
        );
      }
    },
    handleCreateGlobalField() {
      // 打开创建全局系统字段对话框
      this.$refs.customFieldEdit.openGlobalField(
        this.$t('custom_field.create_global_system_field')
      );
    }
  }
}
</script>
```

## ✅ 总结

### 按钮特点

| 特性 | 创建字段 | 创建系统字段 |
|------|---------|-------------|
| **位置** | 右上角 | "创建字段"右侧 |
| **颜色** | 蓝色 | 橙色 |
| **图标** | 无 | 🚩 旗帜 |
| **权限** | 所有用户 | 仅admin |
| **功能** | 创建项目字段 | 创建全局系统字段 |
| **默认值** | 项目字段 | 全局系统字段 |

### 使用建议

1. **日常使用**：点击蓝色"创建字段"按钮
2. **全局字段**：admin点击橙色"创建系统字段"按钮
3. **识别方法**：橙色按钮 + 旗帜图标 = 系统字段功能
