# 一、现代 CLI 工具（替代传统命令）

这些工具用 Rust/Go 重写经典 Unix 命令，更快、更友好。

## 一键安装（Mac）

```bash
brew install ripgrep fd bat eza zoxide fzf tldr jq
```

## 工具清单

| 替代 | 新工具 | 亮点 | 安装 |
|------|--------|------|------|
| `grep` | **ripgrep (rg)** | 快 10-20 倍，自动忽略 .gitignore | `brew install ripgrep` |
| `find` | **fd** | 语法简单，彩色输出 | `brew install fd` |
| `cat` | **bat** | 语法高亮 + 行号 + Git 集成 | `brew install bat` |
| `ls` | **eza** | 图标 + 颜色 + Git 状态 | `brew install eza` |
| `cd` | **zoxide** | 智能跳转，记住常用目录 | `brew install zoxide` |
| `man` | **tldr** | 简明实用示例 | `brew install tldr` |

## 常用示例

```bash
# ripgrep: 在代码中搜索
rg "TODO" --type py           # 只搜 Python 文件
rg "error" -C 3               # 显示上下文 3 行

# fd: 查找文件
fd "\.md$"                    # 找所有 Markdown
fd -e py -x wc -l             # 统计所有 py 文件行数

# bat: 查看文件
bat config.yaml               # 语法高亮
bat -A script.sh              # 显示不可见字符

# eza: 列目录
eza -la --git                 # 长格式 + Git 状态
eza --tree --level=2          # 树形显示 2 层

# zoxide: 智能跳转
z proj                        # 跳到最常用的 *proj* 目录
zi                            # 交互式选择

# fzf: 模糊搜索
vim $(fzf)                    # 模糊查找并打开文件
history | fzf                 # 搜索历史命令
```

## 配置建议

在 `~/.zshrc` 中添加别名：

```bash
alias cat="bat"
alias ls="eza"
alias ll="eza -la --git"
alias tree="eza --tree"
alias grep="rg"
alias find="fd"

# zoxide 初始化
eval "$(zoxide init zsh)"
```

# 二、Python 生态

## uv — 极速包管理器

> 替代 pip/virtualenv/poetry/pyenv，快 10-100 倍
> GitHub: https://github.com/astral-sh/uv ⭐ 60k+

### 安装

```bash
# Mac/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# 或 Homebrew
brew install uv
```

### 核心用法

```bash
# 创建虚拟环境（比 venv 快 80 倍）
uv venv
source .venv/bin/activate

# 安装依赖（比 pip 快 10-100 倍）
uv pip install requests pandas

# 从 requirements.txt 安装
uv pip sync requirements.txt

# 管理 Python 版本
uv python install 3.12
uv python pin 3.12

# 运行脚本（自动处理依赖）
uv run script.py
```

### 项目管理

```bash
# 初始化项目
uv init myproject
cd myproject

# 添加依赖
uv add requests
uv add --dev pytest

# 同步环境
uv sync
```

## Ruff — 极速 Linter

> 替代 Flake8/Black/isort，快 10-100 倍
> GitHub: https://github.com/astral-sh/ruff ⭐ 40k+

```bash
# 安装
brew install ruff

# 检查代码
ruff check .

# 自动修复
ruff check --fix .

# 格式化（替代 Black）
ruff format .
```

# 三、Git/Docker 可视化

## lazygit — Git 终端 UI

> 告别复杂 git 命令，可视化操作
> GitHub: https://github.com/jesseduffield/lazygit ⭐ 55k+

### 安装

```bash
brew install lazygit
```

### 使用

```bash
# 在项目目录启动
lazygit

lg
```

### 核心快捷键

| 按键 | 功能 |
|------|------|

| `c` | 提交 |

| `P` | 拉取 |
| `s` | Squash 提交 |

# 四、开发辅助工具

## it-tools — 开发者瑞士军刀

> 在线工具集：JSON 格式化、Base64、Hash、JWT 解码等
> GitHub: https://github.com/CorentinTh/it-tools ⭐ 26k+

```bash

# 访问 http://localhost:8080

# 或直接用在线版
# https://it-tools.tech
```

包含 80+ 工具：加解密、编码转换、网络工具、文本处理等。

## DevToys — 桌面工具箱（Windows/Mac）

> 离线开发者工具集合
> GitHub: https://github.com/DevToys-app/DevToys ⭐ 28k+

```bash
# Mac
brew install --cask devtoys
```

## jq — JSON 处理神器

```bash
# 安装
brew install jq

# 格式化 JSON
cat data.json | jq .

# 提取字段
cat data.json | jq '.users[].name'

# 过滤
cat data.json | jq '.items | map(select(.price > 100))'
```

## httpie — 更友好的 curl

```bash
# 安装
brew install httpie

# GET 请求
http httpbin.org/get

# POST JSON
http POST api.example.com/users name=john age:=25

# 带 Header
http GET api.example.com Authorization:"Bearer token"
```

# 工具链推荐配置

## 完整安装脚本

```bash
#!/bin/bash
# Mac 开发工具一键安装

# CLI 工具
brew install ripgrep fd bat eza zoxide fzf tldr jq httpie

# Git/Docker
brew install lazygit lazydocker

# Python
curl -LsSf https://astral.sh/uv/install.sh | sh
brew install ruff

# 开发工具
brew install --cask devtoys

echo "安装完成！记得配置 ~/.zshrc"
```

## Shell 配置模板

```bash
# ~/.zshrc 添加

# 现代 CLI 别名
alias cat="bat"
alias ls="eza"
alias ll="eza -la --git"
alias tree="eza --tree"
alias grep="rg"
alias find="fd"
alias lg="lazygit"
alias lzd="lazydocker"

# zoxide
eval "$(zoxide init zsh)"

# fzf
[ -f ~/.fzf.zsh ] && source ~/.fzf.zsh
```
