# WPS JavaScript 宏永久保存方法

## 问题
你的工具代码目前保存在项目文件夹中：
`/Users/edy/ideaProjects/metersphere/docs/ppt-tools/WPS文本颜色工具-零交互版.js`

但每次使用都需要复制粘贴到WPS中，如何让它永久可用？

## 方法一：保存在演示文稿中（推荐）

### 步骤
1. 打开WPS演示
2. 开发工具 → 宏 → JavaScript宏
3. 复制粘贴你的代码
4. **文件 → 另存为 → 选择格式：启用宏的演示文稿 (.pptm)**

### 效果
- ✅ 宏代码保存在 .pptm 文件中
- ✅ 每次打开这个文件，宏都可用
- ✅ 可以分享给其他人（发送 .pptm 文件）

### 使用方式
```
1. 打开 .pptm 文件
2. 开发工具 → 宏 → JavaScript宏
3. 选择函数运行
```

### 适用场景
- 个人使用
- 特定项目使用
- 需要分享给同事

---

## 方法二：创建模板文件

### 步骤
1. 创建一个空白演示文稿
2. 添加你的宏代码
3. 保存为模板：**文件 → 另存为 → WPS演示模板 (.dpt)**
4. 将模板保存到WPS模板目录

### 效果
- ✅ 每次新建演示文稿时可以选择这个模板
- ✅ 新文档自动包含宏代码

### 使用方式
```
1. 新建 → 从模板创建
2. 选择你的模板
3. 宏代码自动可用
```

2. 开发工具 → 宏 → JavaScript宏

## 注意事项

### 安全提示
- .pptm 文件包含宏代码，打开时WPS可能提示安全警告
- 需要启用宏才能运行
- 分享给他人时，告知对方启用宏

### 代码更新
- 如果修改了代码，需要重新保存 .pptm 文件
- 或者在WPS中直接编辑宏代码
