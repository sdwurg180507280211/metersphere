## 🤔 什么是 GeForce NOW？

简单来说，**GeForce NOW 就像一个云端游戏主机**：

- 你不需要高配电脑
- 游戏在 NVIDIA 的云端服务器上运行
- 画面通过串流传输到你的设备上
- 就像在本地玩一样，但不需要高端显卡！

这样你就能用低配电脑、甚至手机玩《黑神话：悟空》了！

## ❓ Steam 云畅玩 vs GeForce NOW

很多小伙伴分不清这两个概念：

| 对比项       | Steam 云畅玩                          | GeForce NOW                            |
| ------------ | ------------------------------------- | -------------------------------------- |
| **本质**     | 对接第三方云游戏服务的平台            | 独立的云游戏服务                        |
| **提供方**   | Valve（Steam）                        | NVIDIA                                 |
| **服务器**   | 不提供，靠第三方                      | 自己在全球部署服务器                    |
| **免费版**   | 没有                                   | 有，但有限制                           |

### 简单理解

```
Steam 云畅玩 = 中介平台
     ↓
GeForce NOW = 实际提供服务的公司
```

## 🛠️ 第二步：注册 GeForce NOW 账号

### 访问官网

打开浏览器访问：https://www.nvidia.com/en-us/geforce-now/

### 选择套餐

点击右上角 **"Join Now"**，会看到三个选项：

| 套餐         | 价格       | 特点                                      |
| ------------ | ---------- | ----------------------------------------- |
| **Free**     | 免费       | 需排队，每次 1 小时会话，720p/60fps     |
| **Priority** | ~$10/月   | 无需排队，6 小时会话，1080p/60fps        |
| **Ultimate** | ~$20/月   | 无需排队，8 小时会话，4K/120fps          |

**推荐先从 Free 版开始尝试！**

### 注册账号

1. 使用邮箱注册（建议 Gmail/Outlook 等国际邮箱）
2. 验证邮箱
3. 设置密码
4. 完成注册！

| 平台         | 下载方式                                  |

## 🛠️ 第四步：连接 Steam 账号

这是最关键的一步！

### 操作步骤

1. 打开 GeForce NOW 客户端
2. 点击右上角头像 → **Settings（设置）**
3. 找到 **Connections（连接）** 或 **Game Library Sync（游戏库同步）**
4. 点击 **Steam** 旁边的 **"Connect"（连接）** 按钮
5. 会弹出 Steam 登录页面
6. 登录你的 Steam 账号（就是买了黑神话的那个！）
7. 授权 GeForce NOW 访问你的游戏库

### 成功后应该看到

Steam 图标旁边会显示 **"Connected"（已连接）** ✓

---

## 🛠️ 第五步：搜索并启动《黑神话：悟空》

### 搜索游戏

1. 在 GeForce NOW 主界面顶部搜索栏
2. 输入：**"Black Myth: Wukong"**
3. 或者输入中文：**"黑神话：悟空"**

### 如果找到了游戏

1. 点击游戏图标
2. 点击 **"Play"（游玩）** 按钮
3. 如果是免费版，可能需要排队（会显示预估等待时间）
4. 排队时可以做别的事，到点了会通知你
5. 连接成功后会自动启动云端 Steam
6. Steam 会自动加载《黑神话：悟空》
7. 开始玩！🎉

### 如果找不到游戏

😭 说明《黑神话：悟空》可能还没在 GeForce NOW 上架

**你可以：**

1. 在 GeForce NOW 论坛请求添加游戏
2. 关注游戏新闻，等官方公告
3. 或者看看备选方案（后面会讲）

## ⚠️ 重要注意事项

### VPN 相关

| 注意事项     | 说明                                      |
| ------------ | ----------------------------------------- |
| **全程开启** | 整个游玩过程中 VPN 不能断开！           |
| **稳定优先** | 选择稳定的节点，不要频繁切换              |
| **IP 一致**  | 注册和游玩最好用同一个地区的 IP           |

### 会话限制

| 套餐         | 单次会话时长 | 排队情况 |
| ------------ | ------------ | -------- |
| **Free**     | 1 小时       | 需要排队 |
| **Priority** | 6 小时       | 无需排队 |
| **Ultimate** | 8 小时       | 无需排队 |

**免费版小贴士：** 快到 1 小时时提前保存存档！断开后可以重新排队继续玩。

### 云存档

**非常重要！** 确保《黑神话：悟空》启用了 Steam 云存档：

1. 在 Steam 中右键点击游戏
2. 属性 → 通用
3. 确认 **"为该游戏启用 Steam 云同步"** 已勾选

这样在云端玩的存档才能同步回来！

### 网络质量

如果遇到延迟高、画面卡顿：

1. 尝试更换 VPN 节点
2. 关闭其他占用带宽的软件
3. 用网线连接电脑（比 Wi-Fi 稳定）
4. 在 GeForce NOW 设置里降低画质

## 🎮 游玩中常见问题

### Q: 画面模糊怎么办？

**A:** 在 GeForce NOW 设置里调整：
- 找到 **"Streaming Quality"（串流质量）**
- 调高 bitrate（比特率）
- 确保网络带宽足够

### Q: 延迟太高感觉操作慢半拍？

**A:**
- 换更近的 VPN 节点（香港 > 日本 > 新加坡 > 美国）
- 用网线代替 Wi-Fi
- 关闭路由器上其他设备

### Q: 每次只能玩 1 小时不够怎么办？

**A:**
- 免费版确实有这个限制
- 可以考虑升级到 Priority 或 Ultimate
- 或者每 1 小时保存后重新排队

### Q: 提示"该地区不支持"？

**A:**
- 检查 VPN 是否真的连上了
- 确认 IP 地址在支持地区
- 换个 VPN 节点试试
