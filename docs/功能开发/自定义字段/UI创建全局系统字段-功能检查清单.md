```sql
SELECT COUNT(*) FROM custom_field_template cft
JOIN issue_template it ON cft.template_id = it.id
WHERE cft.field_id = (
    SELECT id FROM custom_field 
    WHERE name = '测试字段' AND scene = 'ISSUE' AND system = 1 AND global = 1

AND it.global = 1;

-- 3. 检查项目模板关联（应返回项目数量）
SELECT COUNT(*) FROM custom_field_template cft

    WHERE name = '测试字段' AND scene = 'ISSUE' AND system = 1 AND global = 1
)
AND it.global = 0;
```

- ✅ 字段记录存在，`system=1`, `global=1`
