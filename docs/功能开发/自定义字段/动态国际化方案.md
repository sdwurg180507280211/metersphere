# 动态国际化方案 - 解决新增系统字段显示空白问题

## 🔍 问题分析

### 当前问题

1. **静态国际化**：翻译文本硬编码在前端 `zh-CN.js` / `en-US.js` / `zh-TW.js`
2. **显示空白**：新增字段的i18n key在翻译文件中不存在，导致显示为空
3. **部署依赖**：每次新增字段需要修改前端代码、重新编译、重新部署

### 核心矛盾

```
用户需求：通过UI创建字段，立即生效，无需部署
技术现状：国际化写死在前端，必须重新部署才能显示
```

---

## 💡 解决方案对比

### 方案1：直接显示字段名（最简单）⭐⭐⭐⭐⭐

**思路**：用户创建的字段直接显示字段名，不使用国际化

**优点**：
- ✅ 最简单，无需改动架构
- ✅ 立即生效，无需部署
- ✅ 实现成本低（1小时）

**缺点**：
- ⚠️ 不支持多语言切换
- ⚠️ 字段名必须是中文

**适用场景**：
- 公司内部使用，只用中文
- 不需要国际化支持
- 快速实现

---

### 方案2：动态国际化（数据库存储）⭐⭐⭐⭐

**思路**：将翻译文本存储在数据库中，前端动态加载

**优点**：
- ✅ 支持多语言
- ✅ 无需重新部署
- ✅ 用户可在UI中编辑翻译

**缺点**：
- ⚠️ 需要改动架构（1-2天）
- ⚠️ 增加数据库表

**适用场景**：
- 需要国际化支持
- 愿意投入开发时间
- 长期维护

---

### 方案3：混合方案（推荐）⭐⭐⭐⭐⭐

**思路**：
- 系统预定义字段：使用静态国际化
- 用户创建字段：使用动态国际化或直接显示

**优点**：
- ✅ 兼容现有系统
- ✅ 灵活可扩展
- ✅ 开发成本适中

**缺点**：
- ⚠️ 需要区分字段类型

---

## 🚀 推荐实施方案：方案3（混合方案）

### 核心思路

```
字段显示名称的决策逻辑：

1. 判断字段是否是系统预定义字段
   ├── 是 → 使用静态国际化（SYSTEM_FIELD_NAME_MAP）
   └── 否 → 使用动态国际化
       ├── 有翻译 → 显示翻译
       └── 无翻译 → 显示字段名（中文）
```

```java
// CustomFieldController.java

/**
 * 获取字段的国际化翻译
 */
@GetMapping("/i18n/{fieldId}")
public Map<String, String> getFieldI18n(@PathVariable String fieldId) {
    List<CustomFieldI18n> i18nList = customFieldI18nMapper.selectByFieldId(fieldId);
    Map<String, String> result = new HashMap<>();
    for (CustomFieldI18n i18n : i18nList) {
        result.put(i18n.getLocale(), i18n.getFieldName());
    }
    return result;
}

/**
 * 批量获取字段的国际化翻译
 */
@PostMapping("/i18n/batch")
public Map<String, Map<String, String>> batchGetFieldI18n(@RequestBody List<String> fieldIds) {
    Map<String, Map<String, String>> result = new HashMap<>();
    for (String fieldId : fieldIds) {
        List<CustomFieldI18n> i18nList = customFieldI18nMapper.selectByFieldId(fieldId);
        Map<String, String> translations = new HashMap<>();
        for (CustomFieldI18n i18n : i18nList) {
            translations.put(i18n.getLocale(), i18n.getFieldName());
        }
        result.put(fieldId, translations);
    }
    return result;
}

/**
 * 更新字段翻译
 */
@PostMapping("/i18n/update")
@RequiresPermissions(PermissionConstants.SYSTEM_SETTING_READ)
public void updateFieldI18n(@RequestBody CustomFieldI18n i18n) {
    i18n.setUpdateTime(System.currentTimeMillis());
    customFieldI18nMapper.updateByPrimaryKey(i18n);
}
```

### 阶段3：前端实现

#### 3.1 创建动态国际化工具

```javascript
// dynamic-i18n.js
import {get, post} from 'metersphere-frontend/src/plugins/request';

/**
 * 动态国际化管理器
 */
class DynamicI18nManager {
  constructor() {
    this.cache = new Map(); // 缓存：fieldId -> {locale -> translation}
    this.currentLocale = 'zh-CN';
  }
  
  /**
   * 设置当前语言
   */
  setLocale(locale) {
    this.currentLocale = locale;
  }
  
  /**
   * 获取字段翻译
   */
  async getFieldName(fieldId, fieldNameFallback) {
    // 1. 检查缓存
    if (this.cache.has(fieldId)) {
      const translations = this.cache.get(fieldId);
      return translations[this.currentLocale] || fieldNameFallback;
    }
    
    // 2. 从API加载
    try {
      const response = await get(`/custom/field/i18n/${fieldId}`);
      const translations = response.data;
      this.cache.set(fieldId, translations);
      return translations[this.currentLocale] || fieldNameFallback;
    } catch (error) {
      console.error('Failed to load field i18n:', error);
      return fieldNameFallback;
    }
  }
  
  /**
   * 批量加载字段翻译
   */
  async batchLoadFields(fieldIds) {
    if (!fieldIds || fieldIds.length === 0) return;
    
    // 过滤掉已缓存的
    const uncachedIds = fieldIds.filter(id => !this.cache.has(id));
    if (uncachedIds.length === 0) return;
    
    try {
      const response = await post('/custom/field/i18n/batch', uncachedIds);
      const result = response.data;
      
      // 更新缓存
      Object.keys(result).forEach(fieldId => {
        this.cache.set(fieldId, result[fieldId]);
      });
    } catch (error) {
      console.error('Failed to batch load field i18n:', error);
    }
  }
  
  /**
   * 清除缓存
   */
  clearCache() {
    this.cache.clear();
  }
}

export const dynamicI18n = new DynamicI18nManager();
```

#### 3.2 修改字段显示逻辑

```vue
<!-- CustomFieldFormItem.vue - 字段显示组件 -->
<template>
  <div>
    <span>{{ displayName }}</span>
  </div>
</template>

<script>
import {SYSTEM_FIELD_NAME_MAP} from "metersphere-frontend/src/utils/table-constants";
import {dynamicI18n} from "@/utils/dynamic-i18n";

export default {
  props: {
    field: Object,
  },
  data() {
    return {
      displayName: '',
    };
  },
  mounted() {
    this.loadFieldName();
  },
  watch: {
    '$i18n.locale'() {
      this.loadFieldName();
    }
  },
  methods: {
    async loadFieldName() {
      // 1. 判断是否是系统预定义字段
      if (this.field.system && SYSTEM_FIELD_NAME_MAP[this.field.name]) {
        // 系统预定义字段：使用静态国际化
        this.displayName = this.$t(SYSTEM_FIELD_NAME_MAP[this.field.name]);
        return;
      }
      
      // 2. 用户创建的字段：使用动态国际化
      if (this.field.global && this.field.system) {
        // 全局系统字段：从数据库加载翻译
        dynamicI18n.setLocale(this.$i18n.locale);
        this.displayName = await dynamicI18n.getFieldName(
          this.field.id, 
          this.field.name  // fallback：直接显示字段名
        );
      } else {
        // 项目级字段：直接显示字段名
        this.displayName = this.field.name;
      }
    }
  }
};
</script>
```

#### 3.3 优化：在获取字段列表时批量加载翻译

```javascript
// 在获取字段列表的地方
methods: {
  async loadCustomFields() {
    // 1. 获取字段列表
    const response = await this.$get('/custom/field/list', this.condition);
    this.fields = response.data;
    
    // 2. 批量加载翻译（只加载全局系统字段的翻译）
    const globalSystemFieldIds = this.fields
      .filter(f => f.global && f.system)
      .map(f => f.id);
    
    if (globalSystemFieldIds.length > 0) {
      await dynamicI18n.batchLoadFields(globalSystemFieldIds);
    }
  }
}
```

#### 4.1 翻译编辑界面

```vue
<!-- FieldI18nEdit.vue - 翻译编辑对话框 -->
<template>
  <el-dialog 
    title="编辑字段翻译" 
    :visible.sync="visible"
    width="600px">
    
    <el-form :model="form" label-width="100px">
      <el-form-item label="字段名称">
        <el-input :value="fieldName" disabled />
      </el-form-item>
      
      <el-form-item label="中文简体">
        <el-input v-model="form.zhCN" placeholder="请输入中文翻译" />
      </el-form-item>
      
      <el-form-item label="英文">
        <el-input v-model="form.enUS" placeholder="Please enter English translation" />
      </el-form-item>
      
      <el-form-item label="中文繁体">
        <el-input v-model="form.zhTW" placeholder="請輸入繁體中文翻譯" />
      </el-form-item>
    </el-form>
    
    <div slot="footer">
      <el-button @click="visible = false">取消</el-button>
      <el-button type="primary" @click="save">保存</el-button>
    </div>
  </el-dialog>
</template>

<script>
export default {
  data() {
    return {
      visible: false,
      fieldId: '',
      fieldName: '',
      form: {
        zhCN: '',
        enUS: '',
        zhTW: '',
      },
    };
  },
  methods: {
    async open(field) {
      this.visible = true;
      this.fieldId = field.id;
      this.fieldName = field.name;
      
      // 加载现有翻译
      const res = await this.$get(`/custom/field/i18n/${field.id}`);
      const translations = res.data;
      this.form.zhCN = translations['zh-CN'] || '';
      this.form.enUS = translations['en-US'] || '';
      this.form.zhTW = translations['zh-TW'] || '';
    },
    async save() {
      // 保存翻译
      const updates = [
        {fieldId: this.fieldId, locale: 'zh-CN', fieldName: this.form.zhCN},
        {fieldId: this.fieldId, locale: 'en-US', fieldName: this.form.enUS},
        {fieldId: this.fieldId, locale: 'zh-TW', fieldName: this.form.zhTW},
      ];
      
      for (const update of updates) {
        await this.$post('/custom/field/i18n/update', update);
      }
      
      this.$success('保存成功');
      this.visible = false;
      this.$emit('refresh');
      
      // 清除缓存
      dynamicI18n.clearCache();
    }
  }
};
</script>
```
