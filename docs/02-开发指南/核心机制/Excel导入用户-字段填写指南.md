# Excel 导入用户 - 字段填写指南（避坑版）

## 📋 快速参考

### 常见错误速查表

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| "ID不支持中文" | ID包含中文字符 | 只能用英文、数字、下划线 |
| "密码格式不正确" | 密码不符合复杂度 | 必须8-30位，包含大小写字母+数字 |
| "工作空间不存在" | 工作空间名称错误 | 复制系统中的准确名称（区分大小写） |
| "邮箱格式错误" | 邮箱格式不对 | 必须是标准邮箱格式 |
| "ID重复" | Excel中有重复ID | 检查Excel中是否有相同ID |
| "用户ID已存在" | 数据库中已有此ID | 改用其他ID或删除该行（会自动更新） |

## 📝 字段详解（按填写顺序）

### 1. ID（必填）⭐⭐⭐

**字段说明**：用户的唯一标识符，用于登录和系统识别

**填写规则**：
```
✅ 正确示例：
- zhangsan
- zhang_san
- user001
- test_user_2024

❌ 错误示例：
- 张三          （包含中文）
- zhang san     （包含空格）
- user@001      （包含特殊字符@）
- 超过50个字符的ID （长度超限）
```

**验证规则**：
- ✅ 只能包含：英文字母、数字、下划线
- ✅ 长度：1-50 个字符
- ❌ 不能包含：中文、空格、特殊符号
- ❌ 不能重复：Excel内不能重复

**特殊说明**：
- 如果ID在数据库中已存在 → 走**更新流程**（更新权限，不改密码）
- 如果ID不存在 → 走**新增流程**（创建新用户）

---

### 2. 姓名（必填）⭐⭐⭐

**字段说明**：用户的显示名称，会在系统界面显示

**填写规则**：
```
✅ 正确示例：
- 张三
- 李四
- John Smith
- 测试用户01

❌ 错误示例：
- 张          （少于2个字符）
- 超过50个字符的超长名称... （长度超限）
```

**验证规则**：
- ✅ 长度：2-50 个字符
- ✅ 可以包含：中文、英文、数字、空格
- ❌ 不能太短或太长

---

### 3. 电子邮箱（必填）⭐⭐⭐

**字段说明**：用户的登录邮箱，也用于接收系统通知

**填写规则**：
```
✅ 正确示例：
- zhangsan@example.com
- user.test@company.com.cn
- admin_001@test-domain.com

❌ 错误示例：
- zhangsan              （缺少@和域名）
- zhangsan@            （缺少域名）
- @example.com         （缺少用户名）
- zhangsan@example     （域名不完整）
```

**验证规则**：
- ✅ 必须符合标准邮箱格式：`用户名@域名.后缀`
- ✅ 用户名可包含：字母、数字、下划线、点、横线
- ❌ Excel内不能重复
- ❌ 不能与数据库中其他用户的邮箱重复

**⚠️ 重要提示**：
- 新增用户：邮箱不能与已有用户重复
- 更新用户：可以修改邮箱，但不能与其他用户冲突

---

### 4. 密码（必填）⭐⭐⭐

**字段说明**：用户的登录密码

**填写规则**：
```
✅ 正确示例：
- Abc12345
- Test@2024
- Admin123456
- Password1

❌ 错误示例：
- 12345678          （只有数字，缺少字母）
- abcdefgh          （只有小写字母，缺少大写和数字）
- ABCDEFGH          （只有大写字母，缺少小写和数字）
- Abc123            （少于8位）
- Abc1                （少于8位）
```

**验证规则**：
- ✅ 长度：8-30 位
- ✅ 必须同时包含：
  - 至少1个小写字母（a-z）
  - 至少1个大写字母（A-Z）
  - 至少1个数字（0-9）
- ✅ 可以包含特殊字符（如 @#$%）

**⚠️ 重要提示**：
- 新增用户：密码会被 MD5 加密后存储
- 更新用户：**不会修改密码**（即使填写了也不会更新）

---

### 5. 电话（可选）

**字段说明**：用户的联系电话

**填写规则**：
```
✅ 正确示例：
- 13812345678
- 15987654321
- 18600001111
- （留空也可以）

❌ 错误示例：
- 12345678901        （不是1开头）
- 138-1234-5678      （包含横线）
- +86 13812345678    （包含+86和空格）
- 1381234567         （少于11位）
```

**验证规则**：
- ✅ 如果填写，必须是11位手机号
- ✅ 必须以1开头
- ✅ 第二位必须是3-9
- ✅ 可以留空（不填）

## 🔐 权限字段详解

### 权限体系说明

MeterSphere 有5种用户角色，按权限从高到低：

```
系统管理员（最高权限）
    ↓
工作空间管理员（管理工作空间）
    ↓
工作空间成员（使用工作空间）
    ↓
项目管理员（管理项目）
    ↓
项目成员（使用项目）
    ↓
只读用户（只能查看）
```

**⚠️ 关键规则**：
- 一个用户可以同时拥有多种角色
- 权限字段必须填"是"或"否"
- 如果填"是"，对应的工作空间/项目字段可以为空（但不推荐）

---

### 6. 是否是系统管理员（必填）⭐⭐⭐

**字段说明**：系统最高权限，可以管理所有功能

**填写规则**：
```
✅ 只能填写：
- 是
- 否

❌ 不能填写：
- yes/no
- 1/0
- true/false
- Y/N
```

**权限说明**：
- 是 → 拥有系统所有权限（用户管理、系统设置等）
- 否 → 普通用户

**⚠️ 注意**：
- 系统管理员不需要填写工作空间/项目
- 系统管理员自动拥有所有工作空间和项目的访问权限

---

### 7. 是否是工作空间管理员（必填）⭐⭐

**字段说明**：管理指定工作空间的权限

**填写规则**：
```
✅ 只能填写：
- 是
- 否
```

**关联字段**：工作空间管理员应用工作空间

**权限说明**：
- 是 → 可以管理指定工作空间（成员管理、项目管理等）
- 否 → 无工作空间管理权限

---

### 8. 工作空间管理员应用工作空间（可选）⚠️

**字段说明**：指定该用户可以管理哪些工作空间

**填写规则**：
```
✅ 正确示例：

单个工作空间：
测试工作空间

多个工作空间（换行分隔）：
测试工作空间
开发工作空间
生产工作空间

❌ 错误示例：
测试工作空间,开发工作空间    （用逗号分隔，应该换行）
测试工作空间；开发工作空间    （用分号分隔，应该换行）
test workspace              （名称不存在或大小写错误）
```

**验证规则**：
- ✅ 工作空间名称必须在系统中存在
- ✅ 名称必须完全匹配（区分大小写）
- ✅ 多个工作空间用**换行符**分隔（在Excel中按 Alt+Enter 换行）
- ⚠️ 如果"是否是工作空间管理员"填"否"，此字段会被忽略
- ⚠️ 如果"是否是工作空间管理员"填"是"但此字段为空，用户不会有任何工作空间权限

**如何获取准确的工作空间名称**：
1. 登录系统
2. 进入"系统设置" → "工作空间管理"
3. 复制工作空间名称（注意大小写）

---

### 9. 是否是工作空间成员（必填）⭐⭐

**字段说明**：使用指定工作空间的权限（比管理员权限低）

**填写规则**：
```
✅ 只能填写：
- 是
- 否
```

**关联字段**：工作空间成员应用工作空间

**权限说明**：
- 是 → 可以使用指定工作空间（创建项目、查看数据等）
- 否 → 无工作空间使用权限

---

### 10. 工作空间成员应用工作空间（可选）⚠️

**字段说明**：指定该用户可以使用哪些工作空间

**填写规则**：（同"工作空间管理员应用工作空间"）

**⚠️ 注意**：
- 工作空间管理员 ≠ 工作空间成员
- 可以同时是管理员和成员（不同工作空间）
- 例如：A工作空间的管理员 + B工作空间的成员

---

### 11. 是否是项目管理员（必填）⭐⭐

**字段说明**：管理指定项目的权限

**填写规则**：
```
✅ 只能填写：
- 是
- 否
```

**关联字段**：项目管理员应用项目

**权限说明**：
- 是 → 可以管理指定项目（成员管理、测试计划等）
- 否 → 无项目管理权限

---

### 12. 项目管理员应用项目（可选）⚠️

**字段说明**：指定该用户可以管理哪些项目

**填写规则**：
```
✅ 正确示例：

单个项目：
电商系统测试项目

多个项目（换行分隔）：
电商系统测试项目
支付系统测试项目
用户中心测试项目

❌ 错误示例：
电商系统测试项目,支付系统    （用逗号分隔）
test project                （名称不存在）
```

**验证规则**：
- ✅ 项目名称必须在系统中存在
- ✅ 名称必须完全匹配（区分大小写）
- ✅ 多个项目用**换行符**分隔
- ⚠️ 如果"是否是项目管理员"填"否"，此字段会被忽略

**如何获取准确的项目名称**：
1. 登录系统
2. 进入"项目管理" → "项目列表"
3. 复制项目名称（注意大小写）

---

### 13. 是否是项目成员（必填）⭐⭐

**字段说明**：使用指定项目的权限（比管理员权限低）

**填写规则**：
```
✅ 只能填写：
- 是
- 否
```

**关联字段**：项目成员应用项目

---

### 14. 项目成员应用项目（可选）⚠️

**字段说明**：指定该用户可以使用哪些项目

**填写规则**：（同"项目管理员应用项目"）

---

### 15. 是否是只读用户（必填）⭐

**字段说明**：只能查看，不能修改

**填写规则**：
```
✅ 只能填写：
- 是
- 否
```

**关联字段**：只读用户应用项目

**权限说明**：
- 是 → 只能查看指定项目，不能修改
- 否 → 无只读权限

---

### 16. 只读用户应用项目（可选）⚠️

**字段说明**：指定该用户可以查看哪些项目

**填写规则**：（同"项目管理员应用项目"）

## 🚨 常见错误及解决方案

### 错误1：ID包含中文

```
❌ 错误填写：
ID: 张三

✅ 正确填写：
ID: zhangsan
```

### 错误2：密码不符合复杂度

```
❌ 错误填写：
密码: 12345678        （只有数字）
密码: abcdefgh        （只有小写字母）
密码: Abc123          （少于8位）

✅ 正确填写：
密码: Abc12345        （8位，包含大小写字母和数字）
密码: Test@2024       （可以包含特殊字符）
```

### 错误3：工作空间名称错误

```
❌ 错误填写：
工作空间管理员应用工作空间: test workspace    （名称不存在）
工作空间管理员应用工作空间: 测试工作空间,开发工作空间    （用逗号分隔）

✅ 正确填写：
工作空间管理员应用工作空间: 
测试工作空间
开发工作空间
（在Excel中按 Alt+Enter 换行）
```

### 错误4：权限字段填写错误

```
❌ 错误填写：
是否是系统管理员: yes
是否是系统管理员: 1
是否是系统管理员: true

✅ 正确填写：
是否是系统管理员: 是
是否是系统管理员: 否
```

### 错误5：邮箱格式错误

```
❌ 错误填写：
电子邮箱: zhangsan
电子邮箱: zhangsan@
电子邮箱: @example.com

✅ 正确填写：
电子邮箱: zhangsan@example.com
```

## 💡 实用技巧

### 技巧1：如何在Excel中换行

在同一个单元格中输入多个工作空间/项目：

1. 输入第一个名称
2. 按 `Alt + Enter`（Windows）或 `Option + Enter`（Mac）
3. 输入第二个名称
4. 重复步骤2-3

### 技巧2：如何获取准确的工作空间/项目名称

**方法1：从系统复制**
1. 登录系统
2. 进入对应的管理页面
3. 直接复制名称（Ctrl+C）
4. 粘贴到Excel（Ctrl+V）

**方法2：导出现有用户**
1. 在用户管理页面导出现有用户
2. 查看其他用户的工作空间/项目填写
3. 参考相同的格式

### 技巧3：批量导入前先测试

1. 先导入1-2个用户测试
2. 确认无误后再批量导入
3. 避免大批量导入失败

### 技巧4：更新用户权限

如果只想更新用户权限，不创建新用户：

1. 使用已存在的用户ID
2. 填写新的权限配置
3. 密码字段随便填（不会更新）
4. 导入后权限会被完全替换

## 📞 遇到问题怎么办？

### 1. 查看详细错误信息

导入失败后，系统会显示：
- 错误行号
- 具体错误原因
- 错误字段

### 2. 常见问题自查

- [ ] ID是否包含中文或特殊字符？
- [ ] 密码是否符合复杂度要求？
- [ ] 工作空间/项目名称是否准确？
- [ ] 权限字段是否填"是"或"否"？
- [ ] 邮箱格式是否正确？
- [ ] Excel中是否有重复的ID或邮箱？

### 3. 获取帮助

如果以上方法都无法解决，请提供：
- 错误截图
- Excel文件（脱敏后）
- 具体错误提示
