## 常见问题排查

### 问题1：页面404
- **原因**: 路由未正确配置
- **检查**: `workstation/frontend/src/router/modules/workstation.js`
- **解决**: 确认路由已添加

### 问题2：智能提示不显示
- **原因**: 后端API未启动或返回错误
- **检查**: 浏览器控制台Network标签
- **解决**: 启动后端服务，检查API响应

### 问题3：查询无结果
- **原因**: 数据库无数据或JQL语法错误
- **检查**: 
  1. 浏览器控制台查看请求参数
  2. 后端日志查看SQL语句
- **解决**: 检查数据库数据，修正JQL语法

### 问题4：国际化文本未显示
- **原因**: 词条未正确引入
- **检查**: `workstation/frontend/src/i18n/lang/zh-CN.js`
- **解决**: 确认`advanced_search`词条已定义

### 问题5：编译错误
- **原因**: 依赖缺失或语法错误
- **检查**: 终端编译输出
- **解决**: 
  ```bash
  cd workstation/frontend
  npm install
  npm run workstation
  ```
